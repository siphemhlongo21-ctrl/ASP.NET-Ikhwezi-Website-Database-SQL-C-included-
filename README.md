# ASP.NET-Ikhwezi-Website-Database-SQL-C#-included-
One Web AppIkhwezi Driving School – ASP.NET Web Application

This is a production-style ASP.NET Web Application developed for the Ikhwezi Driving School year project. The platform is built using C#, SQL Server, Azure, and features an AI-powered chatbot to support clients in real time.

Overview

The system is structured using ASP.NET Web Forms with a MasterPage + Content Page architecture, ensuring a consistent and scalable user experience for both public-facing and staff-restricted sections.
